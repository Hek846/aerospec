# App Icons

Place app icon files here:
- `app_icon.png` - Full app icon (1024x1024)
- `app_icon_foreground.png` - Adaptive icon foreground (432x432)

To generate icons, run:
```bash
flutter pub run flutter_launcher_icons
```

## Design
- Teal gradient background (#26D0CE)
- White "S" letter centered
